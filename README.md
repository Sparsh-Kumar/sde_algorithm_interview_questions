# SDE Algorithm Interview Questions

A collection of graph algorithm solutions in Python, written for interview prep. Problems are aligned with [takeUforward](https://takeuforward.org/plus/dsa/) DSA material.

---

## Topics

| File        | Topic                          |
| ----------- | ------------------------------ |
| `graphs/problem_1` | Clone graph (deep copy)        |
| `graphs/problem_2` | DFS, BFS from adjacency edges  |
| `graphs/problem_4` | Detect cycle — undirected, DFS |
| `graphs/problem_5` | Detect cycle — undirected, BFS |
| `graphs/problem_6` | Detect cycle — directed, DFS   |
| `graphs/problem_7` | Detect cycle — directed, BFS   |
| `graphs/problem_8` | Topological Sort - directed, DFS|
| `graphs/problem_9` | Topological Sort - directed, BFS|
---

## Run

Python 3.8+ required. No extra dependencies.

```bash
# From project root
python -m graphs.problem_1
python -m graphs.problem_2
# ... etc
```

Optional: use a virtual environment.

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
```
